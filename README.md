# Bank Marketing Analysis

## Descripción
Proyecto final de el curso Programación en Ciencia de Datos, curso de posgrado de la Universidad Nacional del Altiplano Puno. Orientado al análisis descriptivo y modelado predictivo del dataset Bank Marketing del repositorio UCI Machine Learning Repository.

## Objetivos
- Realizar análisis exploratorio de datos
- Identificar patrones relevantes
- Visualizar relaciones entre variables
- Evaluar modelos predictivos

## Dataset
https://archive.ics.uci.edu/dataset/222/bank+marketing
